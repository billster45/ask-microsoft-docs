# Example prompts

Use these in Copilot Chat **Agent** mode.

## Everyday Office & Microsoft 365

- `search Microsoft Learn for all the ways Copilot in Outlook can help me prepare for a meeting`
- `How do I create a Power Automate flow that saves email attachments to SharePoint automatically? search Microsoft Learn`
- `What's the best way to use Copilot in Excel to analyze sales data and spot trends? search Microsoft Learn`
- `How do I set up conditional formatting rules in Excel to highlight overdue tasks? search Microsoft Learn`
- `How do I use XLOOKUP instead of VLOOKUP? search Microsoft Learn`
- `What keyboard shortcuts are available in Microsoft Teams? search Microsoft Learn`

## Azure & Cloud

- `Give me the Azure CLI commands to create an Azure Container App with a managed identity. search Microsoft Learn`
- `I need to understand Azure Functions end-to-end. search Microsoft Learn and deep dive`
- `What are the current Azure OpenAI Service quotas and limits? fetch full doc`
- `How do I set up a CI/CD pipeline with Azure DevOps for a Node.js app? search Microsoft Learn`
- `Find the latest Microsoft Docs on Azure Container Apps managed identity and provide links. search Microsoft Learn`

## Developer / .NET

- `Is gpt-4.1-mini available in EU regions? fetch full doc`
- `Show me the correct way to implement IHttpClientFactory in a .NET 8 minimal API. search Microsoft Learn and fetch full doc`
- `How do I add authentication to an ASP.NET Core web app? search Microsoft Learn and deep dive`
- `What's new in C# 13? search Microsoft Learn`
- `Compare ASP.NET Core Minimal APIs vs Controllers using official docs only. search Microsoft Learn`

## Tips

- Add `search Microsoft Learn` to push Copilot to use Microsoft Learn MCP tools.
- Add `fetch full doc` when you want the complete documentation page.
- Add `deep dive` when you want a more thorough, multi-step answer.
- Ask follow-up questions. Copilot remembers context from previous tool calls.
